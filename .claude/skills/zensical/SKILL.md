## Overview
Zensical is a modern static site generator — built by the creators of Material for MkDocs — that produces beautiful, searchable documentation sites from Markdown files. It is written in Rust and Python, supports both `zensical.toml` and `mkdocs.yml` configuration, and ships with a rich set of Markdown extensions, navigation options, and theming capabilities out of the box.
